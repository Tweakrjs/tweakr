    Tweakr

    Tweakr v1.2 â€“ Latest Release ðŸš€ Release Date: 2025-10-08

    A modern, lightweight utility library for JavaScript/TypeScript. Provides functional, async, array, number, object, browser, and string helpers, fully typed, tree-shakable, and thoroughly tested.

    Installation

    npm

    npm install tweakr

    yarn

    yarn add tweakr

    pnpm

    pnpm add tweakr

    Usage

    ESM / Tree-Shakable

    import { flatten, unique } from 'tweakr';

    console.log(flatten([1,[2,3]])); // [1,2,3]
    console.log(unique([1,2,2,3]));  // [1,2,3]

    CJS / Node.js

    const { flatten, debounce } = require('tweakr');
    console.log(flatten([1,[2,3]])); // [1,2,3]

    Browser Global (UMD)

    <script src="https://cdn.jsdelivr.net/npm/tweakr/dist/tweakr.min.js"></script>
    <script>
      console.log(Tweakr.flatten([1,[2,3]]));
    </script>
